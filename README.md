# SHK Cakal — Redesign

Modernes, schnelles One-Page-Redesign für den Heizung/Sanitär/Klima-Meisterbetrieb SHK Cakal (Salzgitter).
Reines HTML/CSS/JS, Tailwind CSS via CDN, Google Font "Inter"/"Manrope", keine externen Bilder (SVG-Icons, CSS-Gradients).

## Struktur
- `index.html` — Startseite (Hero, Leistungen, Über uns, Ablauf, Kontakt)
- `impressum.html` — Impressum (Platzhalter für fehlende Pflichtangaben)
- `datenschutz.html` — Datenschutzerklärung (Entwurf)
- `assets/css/style.css` — Custom-Styles, Animationen
- `assets/js/main.js` — Mobile-Menü, Scroll-Reveal, Header-Scroll-State
